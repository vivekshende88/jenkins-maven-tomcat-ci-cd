# Jenkins + GitHub + Maven + Tomcat CI/CD Project

This project demonstrates a simple CI/CD pipeline using Jenkins, GitHub, Maven, and Apache Tomcat.

## Workflow
1. Source code is stored in GitHub
2. Jenkins pulls the code automatically
3. Maven builds and packages the application
4. A WAR file is generated
5. The WAR file is deployed to Apache Tomcat

## Tools Used
- GitHub
- Jenkins
- Maven
- Apache Tomcat

## Ports Used
- Jenkins: 8080
- Tomcat: 9090
